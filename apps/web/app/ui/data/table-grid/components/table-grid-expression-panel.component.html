Expreimental Feature!

Behavior and UI may change.

Raw and unformatted property values are used to evaluate the expression. They may be different from what you see in the table view. Refer to the detail view for actual values.

Use the "ignore filter" button on the root group, to disable the filter while you are working on an expression.

Filter is persisted in the URL. If you want to save a state, copy the URL. A proper save feature will be added in the future.

Please report bugs and suggestions on Discord.
